Manage Console - Create new service journey

User journey for the create new service functionality in Manage Console

From: DfE Sign-in Team

Published: 29th September 2023

Last updated: 29th September 2023

Journey overview

Users can search by organisation and can manage and search for users in Manage Console.

Users can now filter organisations by status and organistion category & can filter users by user type, organisation type, last login and account status.

Who can access this journey?

  • Internal DfE users who can access Manage Console
  • Users with the correct permission to create a new service

Current Functionality

  • Filter organisations by status and organisation category
  • Search for organisations
  • Filter users by user type, organisation type, last login and account status
  • Search for users
