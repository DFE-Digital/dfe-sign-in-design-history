Manage Console - Organisation search and user management filtering journey

User journey for the organisation search and user management filtering functionality in Manage Console

From: DfE Sign-in Team

Published: 29th September 2023

Last updated: 29th September 2023

Journey overview

In order to create a new service users currently have to contact DfE Sign-in and complete a manual process in the form of a word document to provide details for the service to be setup.

Users with the correct permission level will be able to create a new service and provide the details required for the DfE Sign-in to review and set up a new service.

Who can access this journey?

  • Internal DfE users who can access Manage Console
  • Users with the correct permission to create a new service

Current Functionality

  • Currently users cannot create a new service via Manage Console it is currently a manual process and requires support from the DfE Sign-in team.
