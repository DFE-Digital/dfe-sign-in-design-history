Manage Console - Manage policies journey

User journey for the Manage policies section of Manage Console

From: DfE Sign-in Team

Published: 29th September 2023

Last updated: 29th September 2023

Journey overview

As part of Manage Console users previously had the ability to view policies. Policies are important as they control who is able to access certain roles/sub-services.

Users with the correct permission level will be able to with the updated functionality be able to: view existing policies, change & edit policies, create new policies & remove policies.

Who can access this journey?

  • Internal DfE users who can access Manage Console
  • Users with the correct permission to Manage policies

Current Functionality

  • Users can view policies,they can view the roles and conditions which are currently assigned to a specific policy.
  • Users are able to request policies to be updated where required by contacting the DfE Sign-in team.
