Manage Console - Service Configuration journey

User journey for the service configuration functionality in Manage Console

From: DfE Sign-in Team

Published: 29th September 2023

Last updated: 29th September 2023

Journey overview

Users can configure and change the setup of their service and how it works in Manage console in DfE Sign-in.

Users will now be able to configure the service with more clarity and understanding of the service configuration functionality and users have less scope to make potentially service outage errors due to the conditional inputs included in the service configuration.

Who can access this journey?

  • Internal DfE users who can access Manage Console
  • Users with the correct permission to configure services

Current Functionality

  • Configure a service and the setup of the backend of the service
