Manage Console - Help section journey

User journey for the help section in Manage Console

From: DfE Sign-in Team

Published: 29th September 2023

Last updated: 29th September 2023

Journey overview

Users can access help and articles which will answer any queries they have about actions they try to complete on the Manage console.

Users can access the help section from multiple different journeys as relevant help links are displayed on the pages which relates to said help link.

Who can access this journey?

  • Internal DfE users who can access Manage Console

Current Functionality

  • Explore the help section through a range of articles on various topics which relate to the DfE Sign-in Manage Console
