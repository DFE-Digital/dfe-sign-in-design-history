MFA - Service create account journey

User journey for when a user creates a new accoutn which requires MFA via the service landing page

From: DfE Sign-in Team

Published: 3rd October 2023

Last updated: 4th October 2023

Journey overview

This journey is for new users to DfE Sign-in who create an account. This journey is only accessible via the individual services login page. They can only access it via the service page as if they create an account via DfE Sign-in directly the service does not know which ervice they require therfore MFA cannot be implemented at this stage.

If a user tries to create an account for a service which uses MFA they will be required to setup MFA as part of the account creation process to be able to access their account.

Who can access this journey?

  • All new users to DfE Sign-in
