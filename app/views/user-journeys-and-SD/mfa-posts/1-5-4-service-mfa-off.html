MFA - Service turns off MFA journey

User journey for when a service turns off MFA for their service

From: DfE Sign-in Team

Published: 3rd October 2023

Last updated: 4th October 2023

Journey overview

MFA is configurable at a service level and each individual service can decide if they wish to use MFA or not.

This journey shows the process of what may happen if a user who accesses a service which requires MFA changes it's MFA configuration and now the user is no longer required to use MFA to login.

Who can access this journey?

  • DfE Sign-in end users who access at least 1 service with MFA switched on
  • All approver users in DfE Sign-in
  • All Manage Console users
  • All Support Console users
