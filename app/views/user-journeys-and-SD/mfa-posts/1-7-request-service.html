MFA - Request new service journey

User journey for when a user requests access to a new service which requires MFA

From: DfE Sign-in Team

Published: 3rd October 2023

Last updated: 4th October 2023

Journey overview

This journey shows the process of an end user who is requesting access to a new service.

In this journey users are warned that the service they wish to access requires MFA and that if their request is approved they will be required to setup MFA the next time they login.

Who can access this journey?

  • DfE Sign-in end users who access at least 1 service with MFA switched on
