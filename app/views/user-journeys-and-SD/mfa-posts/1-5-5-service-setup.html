MFA - Service login & setup MFA journey

User journey for when a user accesses a service via the service landing page and is required to setup MFA

From: DfE Sign-in Team

Published: 3rd October 2023

Last updated: 4th October 2023

Journey overview

This journey shows the journey for when a user accesses a service via the service start page and is required to setup their MFA for the first time as the service has chosen to switch on MFA.

Users will be required to setup MFA as part of the login process and will be able to choose which MFA method they wish to use and can add more than 1 option to their account. The users will be required to use MFA everytime they login after setting this up.

Who can access this journey?

  • DfE Sign-in end users who access at least 1 service with MFA switched on
  • All approver users in DfE Sign-in
  • All Manage Console users
  • All Support Console users
